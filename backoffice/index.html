Connexion GitHub

Ces paramètres restent enregistrés uniquement dans ton navigateur.

Token recommandé

Crée un token GitHub limité au dépôt lucky-reparations-creations avec les droits de lecture/écriture du contenu. Ne partage jamais ce token.

Aide V9

  1. Va dans GitHub → Settings → Developer settings → Personal access tokens.
  2. Crée un token avec accès au dépôt lucky-reparations-creations.
  3. Colle le token dans l'onglet GitHub.
  4. Clique Tester la connexion.
  5. Ajoute ou modifie tes réalisations.
  6. Clique Publier.

Après publication

Cloudflare Pages détecte automatiquement le changement sur GitHub et redéploie le site.
