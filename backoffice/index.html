Token recommandé

Aide V10

  1. Va dans GitHub, crée un token avec Contents: Read and write.
  2. Colle-le dans l'onglet GitHub.
  3. Ajoute ou modifie des réalisations.
  4. Glisse des photos dans les zones prévues.
  5. Clique sur 🚀 Publier.
