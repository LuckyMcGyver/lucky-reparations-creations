Publier sur GitHub

  1. Clique sur Exporter realisations.json.
  2. Va dans GitHub → dépôt lucky-reparations-creations.
  3. Ouvre le dossier content.
  4. Remplace le fichier realisations.json.
  5. Clique Commit changes.
  6. Cloudflare Pages redéploiera automatiquement.
