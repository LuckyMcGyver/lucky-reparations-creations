V5 CMS pour Lucky Réparations & Créations

Important :
Cette version contient Decap CMS (/admin/) pour gérer les réalisations et les photos.
Pour que l'administration publie réellement en ligne, il faut :
1. Mettre le site dans un dépôt GitHub.
2. Connecter ce dépôt à Netlify.
3. Activer Netlify Identity.
4. Activer Git Gateway.
5. Inviter l'adresse e-mail administrateur.
6. Aller sur /admin/ et ajouter les réalisations.

Un simple déploiement ZIP sur Netlify permet d'afficher le site, mais le CMS ne pourra pas enregistrer publiquement sans GitHub + Git Gateway.
